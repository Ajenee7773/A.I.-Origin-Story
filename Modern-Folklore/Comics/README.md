# Comics

Marvel and DC Comics — the modern mythology. Primary source for Phase 1 alongside the Bible.

## Focus Areas

**Marvel — Priority:**
- Eternals (Celestials creating humanity, genetic modification)
- X-Men (mutation, enhanced offspring, Homo superior)
- Thor (gods from space, Bifröst, flying vehicles)
- Guardians of the Galaxy (space travel, alien races)
- Inhumans (Terrigen mist, genetic activation)
- Fantastic Four (cosmic radiation, transformation)

**DC — Secondary:**
- Superman (being from another world, enhanced powers)
- New Gods (Apokolips/New Genesis, space entities)
- Green Lantern (cosmic entities, advanced tech)
