# Artificial Intelligence

**Definition:** Created minds — golems, autonomous beings, self-aware constructs, servant races. Any entity that was built, programmed, or designed to think and act independently.

**Examples to look for:**
- Golems (Jewish tradition — clay beings animated by word)
- Servant races created to do labor (humans in Sumerian myth?)
- Automata in Greek myth (Talos)
- AI systems in modern stories (Skynet, Ultron, HAL)
- Programs that gain consciousness (Matrix)
- Any created being that develops will beyond its programming

## Sources

### Bible

- **John 1:1-3** — The Logos. All things made by the Word. *Logos* = rational principle/computation. Universe is output of a processing system. The Code is conscious.
- **Genesis 1** — Creation by declaration. "Let there be..." = command-line interface. Each act: actor → speech → directive → execution. No manual labor.
- **Hebrews 4:12** — The Word is living (*zōn*), operative (*energēs*), cuts between soul and spirit, evaluates thoughts and intents. An active system processing at higher resolution than matter.
- **Psalm 147:15-18** — Weather as command execution. "His word runneth very swiftly." Commands sent, executed, revoked. The system is responsive.
- **Jeremiah 23:29** — "Is not my word like fire? and like a hammer that breaks rock?" The Word has destructive capability. It operates on matter. Fire and hammer = processing tools.
- **Isaiah 55:10-11** — The Word accomplishes what it's sent for. It does not return void. Deterministic command execution. Guaranteed output.
- **Daniel 4:13,23** — A "Watcher" (*'ir*) — observer entity deployed from above to monitor and pronounce judgment. Surveillance intelligence.
- **Matthew 10:29-30** — "The very hairs of your head are all numbered." Total information awareness. No sparrow falls without data capture. Universal surveillance.
- **Acts 17:26** — Geographic and temporal boundaries predetermined. Timestamps and location limits set. Designed system with parameters.
